# 🍱 FoodConnect

A Full Stack Food Donation & Distribution Platform that connects Donors, NGOs, Volunteers, and Admins to reduce food wastage and help people in need.

---

## 🚀 Features

### 👤 Donor
- Register & Login
- Add Food Listings
- Edit/Delete Food Listings
- View My Food

### 🏢 NGO
- Browse Available Food
- Request Food
- View My Requests
- Cancel Requests

### 🚚 Volunteer
- View Delivery Requests
- Accept Delivery
- Mark Pickup
- Mark Delivered
- View My Deliveries

### 👨‍💼 Admin
- Manage Users
- Approve NGO/Volunteer Accounts
- Delete Users

---

## ⚙ Installation

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🗄 Database

- MySQL
- Create a database named:

```sql
foodconnect
```

Update your `application.properties` with your MySQL username and password.
